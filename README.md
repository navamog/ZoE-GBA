# ZoE-GBA
존 오브 엔더스 GBA 게임 한글패치

로봇게임을 좋아하여 시작한 작업입니다. 비영리 개인작업이며, 이 패치를 사용하시는 분들의 정품구매를 권장합니다.

■ 패치 대상 (반드시 아래 일본판 롬이어야 합니다)
  - 파일:  일본판 롬.gba
  - 크기:  8,388,608 바이트 (8MB)
  - CRC32: 9FEEF66D
  - MD5:   a1ece5ce4fe328f0c0240fc281de4f4b
  - SHA1:  c882179863001f9f2ea15a46ec65da0f465263c1

■ 패치 후 결과물
  - 크기:  8,388,608 바이트 (변화 없음)
  - CRC32: 34EE66F6
  ※ 패치 후 CRC32가 위와 같으면 정상 적용된 것입니다.

----------------------------------------------------------------
■ 패치 방법 (셋 중 편한 것 하나)

  [방법 1] 웹에서 바로 (설치 불필요, 추천)
    1. 브라우저로 https://www.marcrobledo.com/RomPatcher.js/ 접속
    2. "ROM 파일"에 일본판 원본 롬 선택
    3. "Patch 파일"에 ZOE2173_한글패치.ips 선택
    4. [Apply patch] 클릭 → 한글판 롬이 다운로드됨
    ※ 인터넷에 롬이 업로드되지 않고 브라우저 안에서만 처리됩니다.

  [방법 2] Lunar IPS (윈도우)
    1. Lunar IPS(LIPS) 실행 → [Apply IPS Patch]
    2. ZOE2173_한글패치.ips 선택 → 원본 롬 선택
    ※ 원본이 덮어써지므로 미리 복사본을 만들어 두세요.

  [방법 3] Flips (Floating IPS)
    1. flips 실행 → [Apply Patch]
    2. 패치 선택 → 원본 롬 선택 → 저장 이름 지정

----------------------------------------------------------------
■ 주의사항
  - 미국판/유럽판 롬, 헤더가 수정된 롬에는 적용되지 않습니다.
    적용 전 원본 CRC32(9FEEF66D)를 꼭 확인하세요.
  - 기존 세이브(배터리 세이브 .sav)는 호환됩니다.
    단, 에뮬레이터 "세이브스테이트"는 화면 그래픽이 옛 상태로
    남아 있을 수 있으니, 씬 진입 전부터 다시 플레이해야
    새로 수정된 그래픽이 보입니다.

■ 배포 관련
  - 이 패치 파일(.ips)에는 게임 원본 데이터가 포함되어 있지
    않으므로, 패치 파일만 공유하는 것이 팬번역 배포의 관례입니다.
  - 롬 파일 자체를 함께 배포하는 것은 저작권 침해입니다.


※ 파일 사용 전에 반드시 확인 바랍니다.

A. 원칙적으로, 해당기종에 원본 롬 카트리지를 보유한 분이 개인적으로만 이용한다는 전제 하에 본 한글패치 파일을 다운로드할 수 있습니다.

B. 상기 조건(A)에 해당하지 않는 분이 본 한글패치 파일을 다운로드하여 이용할 경우에 발생하는 모든 책임은 이용 당사자에게 있으며, 패치 제작자는 어떠한 책임도 지지 않습니다.

C. 본 한글패치 파일 자체를 제작자에 허가없이 타 사이트에 재배포하지 말아 주시고, 재배포를 원하실 경우 패치원본 게시물의 링크를 공유해 주시기 바랍니다. (아카이빙 포함)

C-1. 단, 제작자의 사망이 확인된 경우에는 예외로 합니다. (제 3자 통보후 3년 혹은 마지막 활동기록후 5년) ﻿ 

D. 모든 저작권은 게임 개발사와 유통사에게 있으며 패치 제작자는 개발사와 유통사에게 어떤 권리도 주장하지 않습니다.
